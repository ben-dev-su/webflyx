# webflyx

Übungsprojekt um Git zu lernen.

## Beschreibung

WebFlyx ist eine fiktive, selbstgehostete Video-Streaming-Anwendung, die Inhalte direkt aus einem Git-Repository ausliefert. Das Repository dient als Übungsumgebung zum Erlernen von Git-Grundlagen: Commits, Branches, Merges, Rebases und die Unterschiede zwischen Plumbing und Porcelain Commands.

## Technologie

- **Werkzeug:** Git
